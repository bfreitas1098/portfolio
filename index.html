About Me

Hi there, I'm Barbara Freitas (as stated above) - A frontend software developer based in south Florida. From an early age, I had a natural curiosity for computers and a keen interest in understanding how software shapes the digital landscape. Two years ago I made the decision to commence my journey as a frontend developer based on my interest in creating web applications that interest and attract users and guide them to the intended product or service being offered. Using tools from my skillset such as HTML5, CSS, SCSS, JavaScript, React, and responsive web design - as well as an understanding of UI design and Figma - I build these web applications.
